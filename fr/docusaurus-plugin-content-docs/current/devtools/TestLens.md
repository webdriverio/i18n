---

TestLens fournit une navigation intelligente entre l'interface DevTools et vos fichiers sources de test. Cliquez sur n'importe quel test dans l'interface pour accéder directement à la définition de ce test dans votre éditeur de code, avec les numéros de ligne exacts.

**Comment ça fonctionne :**
- **Hiérarchie visuelle des tests** - Visualisez la structure de vos tests (suites, tests, scénarios) dans une arborescence
- **Mappage des sources** - Chaque test est lié à son emplacement exact dans vos fichiers source
- **Navigation en un clic** - Cliquez sur n'importe quel test pour l'ouvrir dans votre éditeur
- **Intégration avec l'éditeur** - Fonctionne avec les éditeurs de code populaires via les protocoles file:line

Cette connexion transparente entre l'interface et le code accélère le débogage en éliminant la recherche manuelle de code lors de l'investigation des échecs de tests.
