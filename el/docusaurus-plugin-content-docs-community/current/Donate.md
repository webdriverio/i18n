Εάν η εταιρεία σας χρησιμοποιεί το WebdriverIO και επωφελείται από αυτό, παρακαλούμε ζητήστε από τον διευθυντή σας ή την ομάδα μάρκετινγκ να υποστηρίξει το έργο κάνοντας δωρεά στο συλλογικό ταμείο. Η υποστήριξη θα επιτρέψει στους συντηρητές να αφιερώσουν περισσότερο χρόνο για τη συντήρηση και νέα χαρακτηριστικά για όλους.

Παρακαλούμε επισκεφθείτε τη [σελίδα μας στο OpenCollective](https://opencollective.com/webdriverio) για περισσότερες πληροφορίες σχετικά με το πώς να κάνετε δωρεά στο έργο.

Ευχαριστούμε όλους όσους θα κάνουν ή έχουν ήδη κάνει δωρεά χρημάτων στο συλλογικό ταμείο και συνεπώς έχουν υποστηρίξει το έργο και όλους όσους συνέβαλαν σε αυτό. Σημαίνει πραγματικά πολλά ❤️

## Πολιτικές Εξόδων

Θέλουμε να είμαστε διαφανείς στον τρόπο με τον οποίο αποδεχόμαστε έξοδα στο συλλογικό ταμείο. Ο καθένας θα πρέπει να είναι επιλέξιμος να συμμετέχει και να στέλνει έξοδα για ανάπτυξη συγκεκριμένων χαρακτηριστικών. Θέλουμε επίσης να επιστρέψουμε στην κοινότητα επιτρέποντάς μας να καλύψουμε έξοδα εκδηλώσεων. Οι ακόλουθοι τύποι εξόδων μπορεί να είναι επιλέξιμοι για αποζημίωση από το συλλογικό ταμείο:

### Έξοδα Εκδηλώσεων

Εάν διοργανώνετε μια εκδήλωση που έχει ομιλητή ο οποίος μιλάει για τη χρήση του WebdriverIO και τα χαρακτηριστικά του, μπορείτε να δαπανήσετε έως __$100__. Οι απαιτήσεις αποζημίωσης για έξοδα εκδηλώσεων περιλαμβάνουν:

- Εσείς ή ο λογαριασμός της εκδήλωσης πρέπει να μοιραστείτε το έργο στα μέσα κοινωνικής δικτύωσης (𝕏, Facebook ή LinkedIn) τουλάχιστον 3 φορές
- Η σελίδα της εκδήλωσης πρέπει να έχει το λογότυπο του WebdriverIO και έναν σύνδεσμο προς τη σελίδα του έργου στην περιγραφή της συνάντησής σας
- Πρέπει να χρησιμοποιήσετε τα χρήματα για έξοδα εκδηλώσεων όπως φαγητό, ποτά, ενοικίαση αίθουσας ή εξοπλισμού
- Πρέπει να υποβάλετε αποδείξεις με το αίτημα αποζημίωσής σας

### Έξοδα Ανάπτυξης

Εάν έχετε κάνει αναπτυξιακή εργασία σε οποιοδήποτε από τα αποθετήρια εντός του οργανισμού GitHub του WebdriverIO, μπορείτε να αποζημιωθείτε έως __$1000__ εάν πληρούνται οι ακόλουθες προϋποθέσεις:

- Πρέπει να έχετε υποβάλει επιλέξιμα αιτήματα έλξης (pull requests) που έχουν κλείσει τουλάχιστον 10 ζητήματα που είχαν επισημανθεί με [`Expensable 💸`](https://github.com/webdriverio/webdriverio/labels/Expensable%20%F0%9F%92%B8)
- Κάθε επιπλέον ζήτημα που κλείνει με αυτή την ετικέτα μπορεί να αποζημιωθεί με $100
- Πρέπει να υποβάλετε συνδέσμους σε όλα τα ζητήματα που έχετε κλείσει λόγω των αιτημάτων έλξης σας
- Για να κλείσετε αυτόματα το εισιτήριο, πρέπει να έχετε ένα μήνυμα commit με τη λέξη-κλειδί Fix. Για παράδειγμα, Fix #1234 για να κλείσετε το εισιτήριο #1234.
- Τα Pull Requests πρέπει να συγχωνευτούν από κάποιον από την [κύρια ομάδα](https://github.com/webdriverio/webdriverio/blob/main/AUTHORS.md#tsc-technical-steering-committee). Εάν υπάρχουν πολλά Pull Requests, το μέλος της κύριας ομάδας επιλέγει είτε το πιο πρόσφατο είτε το καλύτερο - είναι στην κρίση του να αποφασίσει τι είναι καλύτερο για το έργο.
- Πρέπει να διεκδικήσετε ένα ζήτημα [`Expensable 💸`](https://github.com/webdriverio/webdriverio/labels/Expensable%20%F0%9F%92%B8) σχολιάζοντας στο νήμα του ζητήματος για να διασφαλίσετε ότι κανείς άλλος δεν εργάζεται στο ίδιο ζήτημα.
- Οποιοσδήποτε συνεισφέρει στο WebdriverIO είναι επιλέξιμος να χρεώσει την εργασία του εάν τα υλοποιημένα χαρακτηριστικά ή οι διορθώσεις σφαλμάτων δεν αποτελούν στόχους μιας εμπορικής εργασίας.

### Έξοδα Ταξιδιού

Εάν είστε μέλος της [Τεχνικής Καθοδηγητικής Επιτροπής](https://github.com/webdriverio/webdriverio/blob/main/AUTHORS.md#tsc-technical-steering-committee), είστε επιλέξιμος να χρεώσετε πτήσεις και διαμονή σε ξενοδοχεία για ταξίδια σε συνέδρια ή συναντήσεις ως μέρος μιας ομιλίας για το WebdriverIO, που δεν πληρώνεται από την ίδια την εκδήλωση ή μια εταιρεία. Μπορείτε να δαπανήσετε έως $500. Οι απαιτήσεις αποζημίωσης για ταξιδιωτικά έξοδα περιλαμβάνουν:

- Πρέπει να στείλετε μια ανάρτηση από τον κύριο λογαριασμό σας στα μέσα κοινωνικής δικτύωσης (π.χ. 𝕏, LinkedIn ή προσωπικό blog) ευχαριστώντας όλους τους συνεισφέροντες του συλλογικού ταμείου μετά την ολοκλήρωση της εκδήλωσης.
- Πρέπει να χρησιμοποιήσετε τα χρήματα για επιλέξιμα ταξιδιωτικά έξοδα, όπως χερσαίες ή αεροπορικές μεταφορές προς την εκδήλωση και διαμονή σε ξενοδοχεία.
- Πρέπει να υποβάλετε αποδείξεις με το αίτημα αποζημίωσής σας.
