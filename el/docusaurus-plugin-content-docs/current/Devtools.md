Η υπηρεσία DevTools παρέχει μια ισχυρή διεπαφή αποσφαλμάτωσης με βάση το πρόγραμμα περιήγησης για εκτελέσεις δοκιμών WebdriverIO. Σας επιτρέπει να οπτικοποιήσετε, να αποσφαλματώσετε και να ελέγχετε τις δοκιμές σας σε πραγματικό χρόνο μέσω μιας διαδραστικής διαδικτυακής εφαρμογής.

## Επισκόπηση

Αυτή η υπηρεσία σας επιτρέπει να:

- **Επανεκτελέσετε δοκιμές επιλεκτικά** - Κάντε κλικ σε οποιαδήποτε περίπτωση δοκιμής ή σουίτα για να την επανεκτελέσετε άμεσα
- **Αποσφαλματώσετε οπτικά** - Δείτε ζωντανές προεπισκοπήσεις του προγράμματος περιήγησης με αυτόματα στιγμιότυπα οθόνης
- **Παρακολουθήσετε την εκτέλεση** - Δείτε λεπτομερή αρχεία καταγραφής εντολών με χρονικές σημάνσεις και αποτελέσματα
- **Παρακολουθήσετε το δίκτυο & την κονσόλα** - Επιθεωρήστε τις κλήσεις API και τα αρχεία καταγραφής JavaScript
- **Περιηγηθείτε στον κώδικα** - Μεταβείτε απευθείας στα αρχεία πηγαίου κώδικα δοκιμών

## Εγκατάσταση

Εγκαταστήστε την υπηρεσία ως εξάρτηση ανάπτυξης:

```sh
npm install --save-dev @wdio/devtools-service
```

## Διαμόρφωση

Προσθέστε την υπηρεσία στη διαμόρφωση του WebDriverIO:

```js
// wdio.conf.js
export const config = {
    // ...
    services: ['devtools'],
    // ...
};
```

### Επιλογές Υπηρεσίας

Διαμορφώστε την υπηρεσία DevTools με αυτές τις επιλογές:

```js
// wdio.conf.js
export const config = {
    // ...
    services: [
        ['devtools', {
            port: 3000,      // Port for the devtools UI (default: 3000)
        }]
    ],
    // ...
};
```

#### Επιλογές

- **port** (number, default: `3000`) - Αριθμός θύρας για τον διακομιστή UI devtools

## Πώς Λειτουργεί

Όταν εκτελείτε τις δοκιμές WebdriverIO με ενεργοποιημένη την υπηρεσία DevTools:

1. Η υπηρεσία ανοίγει ένα παράθυρο προγράμματος περιήγησης στο `http://localhost:3000` (διαμορφώσιμο)
2. Οι δοκιμές σας εκτελούνται κανονικά ενώ το UI του DevTools εμφανίζει ενημερώσεις σε πραγματικό χρόνο
3. Το UI δείχνει την ιεραρχία δοκιμών, την προεπισκόπηση του προγράμματος περιήγησης, το χρονοδιάγραμμα εντολών και τα αρχεία καταγραφής
4. Μετά την ολοκλήρωση των δοκιμών, μπορείτε να κάνετε κλικ σε οποιαδήποτε δοκιμή για να την επανεκτελέσετε μεμονωμένα
5. Οι δοκιμές επανεκτελούνται στην ίδια συνεδρία προγράμματος περιήγησης για ταχύτερη αποσφαλμάτωση

## Χαρακτηριστικά

Εξερευνήστε τα χαρακτηριστικά του DevTools λεπτομερώς:

- **[Interactive Test Rerunning & Visualization](devtools/interactive-test-rerunning)** - Προεπισκοπήσεις προγράμματος περιήγησης σε πραγματικό χρόνο με επανεκτέλεση δοκιμών
- **[Multi-Framework Support](devtools/multi-framework-support)** - Λειτουργεί με Mocha, Jasmine και Cucumber
- **[Console Logs](devtools/console-logs)** - Καταγραφή και επιθεώρηση της εξόδου της κονσόλας του προγράμματος περιήγησης
- **[Network Logs](devtools/network-logs)** - Παρακολούθηση κλήσεων API και δραστηριότητας δικτύου
- **[TestLens](devtools/testlens)** - Πλοήγηση στον πηγαίο κώδικα με έξυπνη πλοήγηση κώδικα
