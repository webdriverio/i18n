title: Διαδραστική Επανεκτέλεση & Οπτικοποίηση Δοκιμών
---

Παρακολουθήστε την εκτέλεση των δοκιμών σας σε πραγματικό χρόνο με ζωντανές προεπισκοπήσεις του προγράμματος περιήγησης και αυτόματη λήψη στιγμιότυπων οθόνης μετά από κάθε εντολή WebDriver. Το UI εμφανίζει ένα πλήρες οπτικό χρονοδιάγραμμα της εκτέλεσης των δοκιμών σας, δείχνοντας την ακριβή κατάσταση του προγράμματος περιήγησης σε κάθε βήμα.

Αφού ολοκληρωθεί η αρχική εκτέλεση των δοκιμών σας, κάντε κλικ σε οποιαδήποτε περίπτωση δοκιμής ή σουίτα στο DevTools UI για να την επανεκτελέσετε άμεσα. Αυτό εξαλείφει την ανάγκη επανεκκίνησης ολόκληρου του test runner ή τροποποίησης του κώδικά σας, επιταχύνοντας δραματικά τους κύκλους αποσφαλμάτωσης.

**Βασικά Χαρακτηριστικά:**
- **Οπτικοποίηση σε πραγματικό χρόνο** - Αυτόματη λήψη στιγμιότυπων οθόνης μετά από κάθε εντολή (click, toBeExisting, κλπ.)
- **Ζωντανή προεπισκόπηση προγράμματος περιήγησης** - Δείτε την τρέχουσα κατάσταση της εφαρμογής σας κατά την εκτέλεση δοκιμών
- **Χρονοδιάγραμμα εντολών** - Προβολή λεπτομερών αρχείων καταγραφής με χρονοσφραγίδες για κάθε ενέργεια
- **Διαδραστική επανεκτέλεση** - Κάντε κλικ σε οποιαδήποτε δοκιμή ή σουίτα για άμεση επανεκτέλεση
- **Εκτέλεση ίδιας συνεδρίας** - Οι δοκιμές επανεκτελούνται στην ίδια συνεδρία του προγράμματος περιήγησης για ταχύτερη ανατροφοδότηση
- **Κυλιόμενο ιστορικό** - Επανεξετάστε οποιοδήποτε σημείο στην εκτέλεση δοκιμών
