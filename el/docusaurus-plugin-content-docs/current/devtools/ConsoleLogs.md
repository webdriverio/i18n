title: Αρχεία Καταγραφής Κονσόλας
---

Καταγράψτε και επιθεωρήστε όλες τις εξόδους της κονσόλας του προγράμματος περιήγησης κατά την εκτέλεση των δοκιμών. Τα DevTools καταγράφουν μηνύματα κονσόλας από την εφαρμογή σας (`console.log()`, `console.warn()`, `console.error()`, `console.info()`, `console.debug()`) καθώς και αρχεία καταγραφής του πλαισίου WebDriverIO με βάση το `logLevel` που έχει ρυθμιστεί στο αρχείο `wdio.conf.ts`.

**Χαρακτηριστικά:**
- Καταγραφή μηνυμάτων κονσόλας σε πραγματικό χρόνο κατά την εκτέλεση δοκιμών
- Αρχεία καταγραφής κονσόλας προγράμματος περιήγησης (log, warn, error, info, debug)
- Αρχεία καταγραφής πλαισίου WebDriverIO φιλτραρισμένα από το ρυθμισμένο `logLevel` (trace, debug, info, warn, error, silent)
- Χρονικές σημάνσεις που δείχνουν ακριβώς πότε καταγράφηκε κάθε μήνυμα
- Αρχεία καταγραφής κονσόλας που εμφανίζονται μαζί με τα βήματα δοκιμής και τα στιγμιότυπα οθόνης του προγράμματος περιήγησης για πλαίσιο

**Διαμόρφωση:**
```js
// wdio.conf.ts
export const config = {
    // Level of logging verbosity: trace | debug | info | warn | error | silent
    logLevel: 'info', // Controls which framework logs are captured
    // ...
};
```

Αυτό διευκολύνει την αποσφαλμάτωση σφαλμάτων JavaScript, την παρακολούθηση της συμπεριφοράς της εφαρμογής και την προβολή των εσωτερικών λειτουργιών του WebDriverIO κατά την εκτέλεση των δοκιμών.
