---
id: globals
title: Καθολικές Μεταβλητές
---

Στα αρχεία των τεστ σας, το WebdriverIO τοποθετεί καθεμία από αυτές τις μεθόδους και αντικείμενα στο καθολικό περιβάλλον. Δεν χρειάζεται να εισάγετε τίποτα για να τα χρησιμοποιήσετε. Ωστόσο, αν προτιμάτε ρητές εισαγωγές, μπορείτε να κάνετε `import { browser, $, $$, expect } from '@wdio/globals'` και να ορίσετε `injectGlobals: false` στη ρύθμιση του WDIO.

Τα ακόλουθα καθολικά αντικείμενα ορίζονται αν δεν έχουν διαμορφωθεί διαφορετικά:

- `browser`: Αντικείμενο [Browser](https://webdriver.io/docs/api/browser) του WebdriverIO
- `driver`: ψευδώνυμο του `browser` (χρησιμοποιείται όταν εκτελούνται δοκιμές για κινητά)
- `multiRemoteBrowser`: ψευδώνυμο του `browser` ή `driver` αλλά ορίζεται μόνο για συνεδρίες [Multiremote](/docs/multiremote)
- `$`: εντολή για ανάκτηση ενός στοιχείου (δείτε περισσότερα στα [API docs](/docs/api/browser/$))
- `$$`: εντολή για ανάκτηση στοιχείων (δείτε περισσότερα στα [API docs](/docs/api/browser/$$))
- `expect`: πλαίσιο ελέγχων ισχυρισμών για το WebdriverIO (δείτε [API docs](/docs/api/expect-webdriverio))

__Σημείωση:__ Το WebdriverIO δεν έχει έλεγχο των χρησιμοποιούμενων frameworks (π.χ. Mocha ή Jasmine) που ορίζουν καθολικές μεταβλητές κατά την εκκίνηση του περιβάλλοντός τους.
