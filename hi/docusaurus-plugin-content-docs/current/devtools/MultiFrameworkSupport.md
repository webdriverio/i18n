title: मल्टी-फ्रेमवर्क सपोर्ट
---

DevTools स्वचालित रूप से Mocha, Jasmine, और Cucumber के साथ बिना किसी फ्रेमवर्क-विशिष्ट कॉन्फ़िगरेशन की आवश्यकता के काम करता है। बस अपने WebDriverIO कॉन्फिग में सर्विस जोड़ें और सभी सुविधाएँ, आप जिस भी टेस्ट फ्रेमवर्क का उपयोग कर रहे हैं, उससे निर्बाध रूप से काम करेंगी।

**समर्थित फ्रेमवर्क:**
- **Mocha** - ग्रेप फ़िल्टरिंग के साथ टेस्ट और सूइट-स्तरीय निष्पादन
- **Jasmine** - ग्रेप-आधारित फ़िल्टरिंग के साथ पूर्ण एकीकरण
- **Cucumber** - फीचर:लाइन लक्ष्यीकरण के साथ सिनेरियो और उदाहरण-स्तरीय निष्पादन

वही डीबगिंग इंटरफेस, टेस्ट पुनः चलाना, और विज़ुअलाइज़ेशन सुविधाएँ सभी फ्रेमवर्क पर लगातार काम करती हैं।

## कॉन्फ़िगरेशन

```js
// wdio.conf.js
export const config = {
    framework: 'mocha', // या 'jasmine' या 'cucumber'
    services: ['devtools'],
    // ...
};
```
