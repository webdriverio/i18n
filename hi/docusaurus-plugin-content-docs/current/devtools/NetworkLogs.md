---

अपने परीक्षणों के दौरान सभी नेटवर्क गतिविधि की निगरानी और निरीक्षण करें। DevTools हर HTTP अनुरोध और प्रतिक्रिया को कैप्चर करता है, जो आपको API कॉल, संसाधन लोडिंग और नेटवर्क टाइमिंग की पूरी दृश्यता देता है - बिल्कुल ब्राउज़र DevTools की तरह।

**जो कैप्चर किया जाता है:**
- **अनुरोध विवरण** - URL, विधि, हेडर्स, क्वेरी पैरामीटर्स, अनुरोध बॉडी
- **प्रतिक्रिया डेटा** - स्टेटस कोड, प्रतिक्रिया हेडर्स, प्रतिक्रिया बॉडी, टाइमिंग
- **संसाधन प्रकार** - XHR/Fetch अनुरोध, स्क्रिप्ट, स्टाइलशीट, छवियां और अधिक
- **प्रदर्शन मेट्रिक्स** - अनुरोध टाइमिंग, अवधि और नेटवर्क वॉटरफॉल

यह API समस्याओं के डिबगिंग, धीमे अनुरोधों की पहचान, डेटा पेलोड के सत्यापन और परीक्षण के दौरान आपके एप्लिकेशन के नेटवर्क व्यवहार को समझने के लिए अमूल्य है।
