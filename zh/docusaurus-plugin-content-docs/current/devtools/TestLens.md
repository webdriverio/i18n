title: 测试镜像
---

TestLens 提供了 DevTools UI 和测试源文件之间的智能代码导航。点击 UI 中的任何测试可以直接跳转到代码编辑器中该测试的定义处，包含精确的行号。

**工作原理：**
- **可视化测试层次结构** - 在树状视图中查看测试结构（套件、测试、场景）
- **源代码映射** - 每个测试链接到其在源文件中的确切位置
- **一键导航** - 点击任何测试在编辑器中打开它
- **编辑器集成** - 通过 file:line 协议与流行的代码编辑器配合使用

UI 和代码之间的无缝连接加速了调试过程，在调查测试失败时无需手动搜索代码。
