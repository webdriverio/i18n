title: 网络日志
---

在测试过程中监控和检查所有网络活动。DevTools 捕获每个 HTTP 请求和响应，让您完全了解 API 调用、资源加载和网络时序 - 就像浏览器 DevTools 一样。

**捕获内容包括：**
- **请求详情** - URL、方法、标头、查询参数、请求体
- **响应数据** - 状态码、响应标头、响应体、时序
- **资源类型** - XHR/Fetch 请求、脚本、样式表、图片等
- **性能指标** - 请求时序、持续时间和网络瀑布图

这对于调试 API 问题、识别慢请求、验证数据负载以及了解测试期间应用程序的网络行为非常有价值。
