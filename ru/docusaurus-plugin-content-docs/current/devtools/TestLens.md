TestLens предоставляет интеллектуальную навигацию по коду между интерфейсом DevTools и исходными файлами ваших тестов. Нажмите на любой тест в пользовательском интерфейсе, чтобы перейти непосредственно к определению этого теста в вашем редакторе кода, с точными номерами строк.

**Как это работает:**
- **Визуальная иерархия тестов** - Просматривайте структуру ваших тестов (наборы, тесты, сценарии) в виде дерева
- **Сопоставление с исходным кодом** - Каждый тест связан с его точным местоположением в исходных файлах
- **Навигация в один клик** - Нажмите на любой тест, чтобы открыть его в вашем редакторе
- **Интеграция с редактором** - Работает с популярными редакторами кода через протоколы file:line

Эта безупречная связь между пользовательским интерфейсом и кодом ускоряет отладку, устраняя необходимость ручного поиска кода при исследовании сбоев тестов.
