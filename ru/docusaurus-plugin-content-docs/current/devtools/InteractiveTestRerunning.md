title: Интерактивный перезапуск и визуализация тестов
---

Наблюдайте за выполнением ваших тестов в реальном времени с предварительным просмотром браузера и автоматическими скриншотами, которые делаются после каждой команды WebDriver. Интерфейс отображает полную визуальную временную шкалу выполнения теста, показывая точное состояние браузера на каждом шаге.

После завершения первоначального запуска теста щелкните на любом тестовом примере или наборе в интерфейсе DevTools, чтобы мгновенно выполнить его повторно. Это устраняет необходимость перезапускать весь тестовый раннер или изменять ваш код, значительно ускоряя циклы отладки.

**Ключевые особенности:**
- **Визуализация в реальном времени** - Автоматическое создание скриншотов после каждой команды (click, toBeExisting и т.д.)
- **Предварительный просмотр браузера в реальном времени** - Видите текущее состояние вашего приложения во время выполнения теста
- **Хронология команд** - Подробные журналы с метками времени для каждого действия
- **Интерактивный перезапуск** - Щелкните любой тест или набор тестов для мгновенного повторного выполнения
- **Выполнение в той же сессии** - Тесты перезапускаются в той же сессии браузера для более быстрой обратной связи
- **Прокручиваемая история** - Просмотр любого момента выполнения теста
