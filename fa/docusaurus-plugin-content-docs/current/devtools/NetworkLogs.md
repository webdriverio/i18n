---

نظارت و بررسی تمام فعالیت‌های شبکه در طول تست‌های خود. DevTools تمام درخواست‌ها و پاسخ‌های HTTP را ضبط می‌کند و به شما دید کاملی از فراخوانی‌های API، بارگیری منابع و زمان‌بندی شبکه می‌دهد - درست مانند DevTools مرورگر.

**چه چیزی ضبط می‌شود:**
- **جزئیات درخواست** - URL، متد، هدرها، پارامترهای پرس‌وجو، بدنه درخواست
- **داده‌های پاسخ** - کد وضعیت، هدرهای پاسخ، بدنه پاسخ، زمان‌بندی
- **انواع منابع** - درخواست‌های XHR/Fetch، اسکریپت‌ها، برگه‌های سبک، تصاویر و موارد دیگر
- **معیارهای عملکرد** - زمان‌بندی درخواست، مدت زمان و آبشار شبکه

این برای اشکال‌زدایی مشکلات API، شناسایی درخواست‌های کند، اعتبارسنجی محموله‌های داده و درک رفتار شبکه برنامه شما در طول تست‌ها بسیار ارزشمند است.
