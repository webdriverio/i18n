---

DevTools به طور خودکار با Mocha، Jasmine و Cucumber کار می‌کند بدون نیاز به تنظیمات خاص فریم‌ورک. به سادگی سرویس را به پیکربندی WebDriverIO خود اضافه کنید و تمام ویژگی‌ها بدون توجه به اینکه از کدام فریم‌ورک تست استفاده می‌کنید، به طور یکپارچه کار می‌کنند.

**فریم‌ورک‌های پشتیبانی شده:**
- **Mocha** - اجرای سطح تست و سوئیت با فیلترکردن grep
- **Jasmine** - ادغام کامل با فیلترکردن مبتنی بر grep
- **Cucumber** - اجرای سطح سناریو و مثال با هدف‌گذاری feature:line

رابط اشکال‌زدایی، اجرای مجدد تست و ویژگی‌های تجسم به طور یکسان در تمام فریم‌ورک‌ها کار می‌کنند.

## پیکربندی

```js
// wdio.conf.js
export const config = {
    framework: 'mocha', // or 'jasmine' or 'cucumber'
    services: ['devtools'],
    // ...
};
```
