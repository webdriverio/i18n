سرویس DevTools یک رابط اشکال‌زدایی مبتنی بر مرورگر قدرتمند برای اجرای تست‌های WebdriverIO فراهم می‌کند. این به شما امکان می‌دهد تا تست‌های خود را به صورت بصری، اشکال‌زدایی و کنترل در زمان واقعی از طریق یک برنامه تحت وب تعاملی انجام دهید.

## بررسی کلی

این سرویس به شما امکان می‌دهد:

- **اجرای مجدد انتخابی تست‌ها** - روی هر مورد آزمون یا مجموعه کلیک کنید تا فوراً آن را مجدداً اجرا کنید
- **اشکال‌زدایی بصری** - مشاهده پیش‌نمایش زنده مرورگر با تصاویر خودکار
- **پیگیری اجرا** - مشاهده گزارش‌های دستورات با جزئیات همراه با زمان و نتایج
- **نظارت بر شبکه و کنسول** - بررسی فراخوانی‌های API و گزارش‌های جاوااسکریپت
- **پیمایش به کد** - پرش مستقیم به فایل‌های منبع تست

## نصب

سرویس را به عنوان وابستگی توسعه نصب کنید:

```sh
npm install --save-dev @wdio/devtools-service
```

## پیکربندی

سرویس را به پیکربندی WebDriverIO خود اضافه کنید:

```js
// wdio.conf.js
export const config = {
    // ...
    services: ['devtools'],
    // ...
};
```

### گزینه‌های سرویس

سرویس DevTools را با این گزینه‌ها پیکربندی کنید:

```js
// wdio.conf.js
export const config = {
    // ...
    services: [
        ['devtools', {
            port: 3000,      // پورت برای رابط کاربری devtools (پیش‌فرض: 3000)
        }]
    ],
    // ...
};
```

#### گزینه‌ها

- **port** (عدد، پیش‌فرض: `3000`) - شماره پورت برای سرور رابط کاربری devtools

## نحوه کارکرد

وقتی تست‌های WebdriverIO خود را با سرویس DevTools فعال اجرا می‌کنید:

1. سرویس یک پنجره مرورگر در `http://localhost:3000` (قابل تنظیم) باز می‌کند
2. تست‌های شما به طور معمول اجرا می‌شوند در حالی که رابط کاربری DevTools به‌روزرسانی‌های زنده را نمایش می‌دهد
3. رابط کاربری سلسله مراتب تست، پیش‌نمایش مرورگر، جدول زمانی دستورات و گزارش‌ها را نشان می‌دهد
4. پس از اتمام تست‌ها، می‌توانید روی هر تست کلیک کنید تا به صورت جداگانه دوباره اجرا شود
5. تست‌ها در همان نشست مرورگر مجدداً اجرا می‌شوند تا اشکال‌زدایی سریع‌تر شود
