TestLens bietet intelligente Code-Navigation zwischen der DevTools-Benutzeroberfläche und Ihren Test-Quelldateien. Klicken Sie auf einen beliebigen Test in der Benutzeroberfläche, um direkt zur Definition dieses Tests in Ihrem Code-Editor zu springen, komplett mit genauen Zeilennummern.

**Wie es funktioniert:**
- **Visuelle Test-Hierarchie** - Sehen Sie Ihre Teststruktur (Suites, Tests, Szenarien) in einer Baumansicht
- **Source Mapping** - Jeder Test verlinkt zu seiner genauen Position in Ihren Quelldateien
- **Navigation mit einem Klick** - Klicken Sie auf einen beliebigen Test, um ihn in Ihrem Editor zu öffnen
- **Editor-Integration** - Funktioniert mit beliebten Code-Editoren durch file:line-Protokolle

Diese nahtlose Verbindung zwischen Benutzeroberfläche und Code beschleunigt das Debugging, indem die manuelle Codesuche bei der Untersuchung von Testfehlern entfällt.
