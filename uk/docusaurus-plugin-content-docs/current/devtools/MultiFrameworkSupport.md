title: Підтримка кількох фреймворків
---

DevTools автоматично працює з Mocha, Jasmine та Cucumber без необхідності налаштування під конкретний фреймворк. Просто додайте сервіс до вашої конфігурації WebDriverIO, і всі функції працюватимуть безперешкодно незалежно від того, який тестовий фреймворк ви використовуєте.

**Підтримувані фреймворки:**
- **Mocha** - Виконання на рівні тесту та набору тестів з фільтрацією за допомогою grep
- **Jasmine** - Повна інтеграція з фільтрацією на основі grep
- **Cucumber** - Виконання на рівні сценарію та прикладу з націлюванням feature:line

Однаковий інтерфейс налагодження, повторне запускання тестів та функції візуалізації працюють послідовно на всіх фреймворках.

## Конфігурація

```js
// wdio.conf.js
export const config = {
    framework: 'mocha', // або 'jasmine' або 'cucumber'
    services: ['devtools'],
    // ...
};
```
