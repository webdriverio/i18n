---
id: globals
title: குளோபல்கள்
---

உங்கள் சோதனை கோப்புகளில், WebdriverIO இந்த முறைகள் மற்றும் பொருட்கள் ஒவ்வொன்றையும் உலகளாவிய சூழலில் வைக்கிறது. அவற்றைப் பயன்படுத்த நீங்கள் எதையும் இறக்குமதி செய்ய வேண்டியதில்லை. இருப்பினும், நீங்கள் வெளிப்படையான இறக்குமதிகளை விரும்பினால், `import { browser, $, $$, expect } from '@wdio/globals'` செய்யலாம் மற்றும் உங்கள் WDIO கட்டமைப்பில் `injectGlobals: false` என அமைக்கலாம்.

வேறு விதமாக உள்ளமைக்கப்படவில்லை என்றால் பின்வரும் உலகளாவிய பொருட்கள் அமைக்கப்படுகின்றன:

- `browser`: WebdriverIO [Browser object](https://webdriver.io/docs/api/browser)
- `driver`: `browser`-க்கான மற்றொரு பெயர் (மொபைல் சோதனைகளை இயக்கும்போது பயன்படுத்தப்படுகிறது)
- `multiRemoteBrowser`: `browser` அல்லது `driver`-க்கான மற்றொரு பெயர், ஆனால் [Multiremote](/docs/multiremote) அமர்வுகளுக்கு மட்டுமே அமைக்கப்படுகிறது
- `$`: ஒரு எலிமென்ட்டை பெறுவதற்கான கட்டளை (மேலும் [API docs](/docs/api/browser/$) இல் காணலாம்)
- `$$`: எலிமென்ட்களை பெறுவதற்கான கட்டளை (மேலும் [API docs](/docs/api/browser/$$) இல் காணலாம்)
- `expect`: WebdriverIO-க்கான உறுதிப்படுத்தல் கட்டமைப்பு (பார்க்க [API docs](/docs/api/expect-webdriverio))

__குறிப்பு:__ பயன்படுத்தப்படும் கட்டமைப்புகள் (எ.கா. Mocha அல்லது Jasmine) தங்கள் சூழலை துவக்கும் போது உலகளாவிய மாறிகளை அமைப்பதில் WebdriverIO-க்கு எந்த கட்டுப்பாடும் இல்லை.
