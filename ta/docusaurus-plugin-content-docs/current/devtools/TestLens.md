டெஸ்ட்லென்ஸ் உங்கள் டெவ்டூல்ஸ் UI மற்றும் உங்கள் சோதனை மூல கோப்புகளுக்கு இடையே புத்திசாலித்தனமான குறியீட்டு வழிசெலுத்தலை வழங்குகிறது. UI இல் உள்ள எந்தவொரு சோதனையிலும் கிளிக் செய்து, துல்லியமான வரி எண்களுடன் உங்கள் குறியீடு எடிட்டரில் அந்த சோதனையின் வரையறைக்கு நேரடியாகச் செல்லவும்.

**இது எப்படி செயல்படுகிறது:**
- **விஷுவல் டெஸ்ட் படிநிலை** - உங்கள் சோதனை அமைப்பை (சூட்டுகள், சோதனைகள், சூழல்கள்) ஒரு மரக் காட்சியில் பார்க்கலாம்
- **மூல மேப்பிங்** - ஒவ்வொரு சோதனையும் உங்கள் மூல கோப்புகளில் அதன் துல்லியமான இடத்துடன் இணைக்கப்பட்டுள்ளது
- **ஒரு கிளிக் வழிசெலுத்தல்** - எந்த சோதனையிலும் கிளிக் செய்து அதை உங்கள் எடிட்டரில் திறக்கலாம்
- **எடிட்டர் ஒருங்கிணைப்பு** - file:line நெறிமுறைகள் மூலம் பிரபலமான குறியீடு எடிட்டர்களுடன் செயல்படுகிறது

UI மற்றும் குறியீட்டிற்கு இடையேயான இந்த இடைவெளியற்ற இணைப்பு சோதனை தோல்விகளை விசாரிக்கும்போது கையேடு குறியீடு தேடுவதை நீக்குவதன் மூலம் பிழைத்திருத்தத்தை விரைவுபடுத்துகிறது.
