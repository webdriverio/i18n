உங்கள் சோதனைகளின் போது அனைத்து நெட்வொர்க் செயல்பாடுகளையும் கண்காணித்து ஆய்வு செய்யுங்கள். DevTools ஒவ்வொரு HTTP கோரிக்கை மற்றும் பதிலையும் கைப்பற்றுகிறது, API அழைப்புகள், வள ஏற்றம் மற்றும் நெட்வொர்க் நேரம் பற்றிய முழுமையான தெளிவுத்தன்மையை உங்களுக்கு வழங்குகிறது - பிரவுசர் DevTools போலவே.

**கைப்பற்றப்படுவது:**
- **கோரிக்கை விவரங்கள்** - URL, முறை, தலைப்புகள், வினவல் அளவுருக்கள், கோரிக்கை உடல்
- **பதில் தரவு** - நிலை குறியீடு, பதில் தலைப்புகள், பதில் உடல், நேரம்
- **வள வகைகள்** - XHR/Fetch கோரிக்கைகள், ஸ்கிரிப்ட்கள், ஸ்டைல்ஷீட்கள், படங்கள், மற்றும் பல
- **செயல்திறன் அளவீடுகள்** - கோரிக்கை நேரம், காலம், மற்றும் நெட்வொர்க் வாட்டர்ஃபால்

API சிக்கல்களைத் தீர்ப்பதற்கும், மெதுவான கோரிக்கைகளை அடையாளம் காண்பதற்கும், தரவு பேலோடுகளைச் சரிபார்ப்பதற்கும், சோதனைகளின் போது உங்கள் பயன்பாட்டின் நெட்வொர்க் நடத்தையைப் புரிந்துகொள்வதற்கும் இது மதிப்புமிக்கதாகும்.
