---

உங்கள் சோதனைகள் உலவி முன்னோட்டங்களுடன் நிகழ்நேரத்தில் செயல்படுவதைக் காணுங்கள் மற்றும் ஒவ்வொரு WebDriver கட்டளைக்குப் பிறகும் தானாகவே திரைப்பிடிப்புகள் எடுக்கப்படும். UI உங்கள் சோதனை செயலாக்கத்தின் முழுமையான காட்சி காலவரிசையைக் காட்டுகிறது, ஒவ்வொரு படியிலும் உலவியின் துல்லியமான நிலையைக் காட்டுகிறது.

உங்கள் ஆரம்ப சோதனை ஓட்டம் முடிந்தவுடன், DevTools UI-இல் எந்த சோதனை வழக்கையோ அல்லது தொகுப்பையோ க்ளிக் செய்து உடனடியாக மீண்டும் இயக்கலாம். இது உங்கள் முழு சோதனை இயக்கியை மறுதொடக்கம் செய்ய அல்லது உங்கள் குறியீட்டை மாற்ற வேண்டிய அவசியத்தை நீக்குகிறது, பிழைத்திருத்த சுழற்சிகளை குறிப்பிடத்தக்க அளவில் விரைவுபடுத்துகிறது.

**முக்கிய அம்சங்கள்:**
- **நிகழ்நேர காட்சிப்படுத்துதல்** - ஒவ்வொரு கட்டளைக்குப் பிறகும் தானியங்கி திரைப்பிடிப்பு (click, toBeExisting, போன்றவை)
- **நேரடி உலவி முன்னோட்டம்** - சோதனை செயலாக்கத்தின் போது உங்கள் பயன்பாட்டின் தற்போதைய நிலையைக் காணுங்கள்
- **கட்டளை காலவரிசை** - ஒவ்வொரு செயலுக்கும் நேர முத்திரைகளுடன் விரிவான பதிவுகளைக் காணுங்கள்
- **தொடர்புடைய மறு இயக்கம்** - எந்த சோதனை அல்லது தொகுப்பையும் க்ளிக் செய்து உடனடியாக மீண்டும் இயக்கலாம்
- **அதே-அமர்வு செயலாக்கம்** - சோதனைகள் விரைவான கருத்துக்களுக்காக அதே உலவி அமர்வில் மீண்டும் இயக்கப்படுகின்றன
- **உருட்டக்கூடிய வரலாறு** - சோதனை செயலாக்கத்தில் எந்த புள்ளியையும் மதிப்பாய்வு செய்யுங்கள்
