title: பல-பிரேம்வொர்க் ஆதரவு
---

DevTools தானாகவே மோச்சா, ஜாஸ்மின், மற்றும் குக்கும்பர் ஆகியவற்றுடன் எந்தவொரு பிரேம்வொர்க் குறிப்பிட்ட கான்ஃபிகரேஷன் தேவைப்படாமல் செயல்படுகிறது. உங்கள் WebDriverIO கான்ஃபிக்கில் சேவையைச் சேர்க்க வேண்டியது மட்டுமே, நீங்கள் எந்த டெஸ்ட் பிரேம்வொர்க்கைப் பயன்படுத்துகிறீர்கள் என்பதைப் பொருட்படுத்தாமல் அனைத்து அம்சங்களும் தடையின்றி செயல்படும்.

**ஆதரிக்கப்படும் பிரேம்வொர்க்குகள்:**
- **மோச்சா** - கிரெப் வடிகட்டுதலுடன் டெஸ்ட் மற்றும் சூட்-நிலை செயல்பாடு
- **ஜாஸ்மின்** - கிரெப் அடிப்படையிலான வடிகட்டுதலுடன் முழுமையான ஒருங்கிணைப்பு
- **குக்கும்பர்** - feature:line இலக்குடன் சிறப்பம்சம் மற்றும் உதாரண-நிலை செயல்பாடு

அதே டிபக்கிங் இடைமுகம், டெஸ்ட் மறுஇயக்கம், மற்றும் காட்சிப்படுத்தல் அம்சங்கள் அனைத்து பிரேம்வொர்க்குகளிலும் ஒரே மாதிரியாக செயல்படுகின்றன.

## கான்ஃபிகரேஷன்

```js
// wdio.conf.js
export const config = {

    services: ['devtools'],
    // ...
```
