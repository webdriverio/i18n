あなたの会社がWebdriverIOを使用し、その恩恵を受けている場合は、マネージャーやマーケティングチームにコレクティブへの寄付によるプロジェクトのサポートを依頼してください。支援により、メンテナーはメンテナンスや新機能の開発により多くの時間を割くことができます。

プロジェクトへの寄付方法についての詳細は、[OpenCollectiveページ](https://opencollective.com/webdriverio)をご確認ください。

コレクティブにお金を寄付してくださった方、またはこれから寄付してくださる方々、そしてプロジェクトに貢献してくださったすべての方々に感謝いたします。本当に意味のあることです ❤️

## 経費ポリシー

私たちはコレクティブへの経費の受け入れ方法において透明性を保ちたいと考えています。誰もが特定の機能の開発のための経費を申請する資格があるべきです。また、イベント経費の申請を可能にすることでコミュニティに還元したいと考えています。以下の経費タイプがコレクティブから払い戻される資格があります：

### イベント経費

WebdriverIOとその機能について話すスピーカーがいるイベントを主催する場合、最大で__$100__まで経費申請できます。イベント経費の払い戻し要件は以下の通りです：

- あなたまたはイベントアカウントは、ソーシャルメディア（𝕏、FacebookまたはLinkedIn）でプロジェクトを少なくとも3回共有すること
- イベントページにWebdriverIOのロゴとプロジェクトページへのリンクを掲載すること
- 資金は食べ物、飲み物、会場またはイベント機器のレンタルなどの適格なイベント経費に使用すること
- 払い戻しリクエストには領収書を添付すること

### 開発経費

GitHub WebdriverIO組織内のリポジトリで開発作業を行った場合、以下の要件を満たせば最大__$1000__まで払い戻しを受けられます：

- [`Expensable 💸`](https://github.com/webdriverio/webdriverio/labels/Expensable%20%F0%9F%92%B8)ラベルが付けられた少なくとも10個の問題を閉じるプルリクエストを提出していること
- そのラベルが付いた追加の問題を閉じるごとに$100を経費申請できる
- プルリクエストによって閉じたすべての問題へのリンクを提出する必要がある
- チケットを自動的に閉じるには、Fixキーワードを含むコミットメッセージが必要です。例えば、Fix #1234でチケット#1234を閉じます。
- プルリクエストは[コアチーム](https://github.com/webdriverio/webdriverio/blob/main/AUTHORS.md#tsc-technical-steering-committee)のメンバーによってマージされる必要があります。複数のプルリクエストがある場合、コアチームメンバーは最新のものまたは最良のものを選択します - プロジェクトに最適なものを決定するのは彼らの判断です。
- 他の人が同じ問題に取り組まないように、[`Expensable 💸`](https://github.com/webdriverio/webdriverio/labels/Expensable%20%F0%9F%92%B8)の問題に対してコメントすることで主張する必要があります。
- 実装された機能やバグ修正が商業的な仕事の目的でない場合、WebdriverIOに貢献する人は誰でも自分の作業を経費申請する資格があります。

### 旅費

[技術運営委員会チーム](https://github.com/webdriverio/webdriverio/blob/main/AUTHORS.md#tsc-technical-steering-committee)のメンバーである場合、WebdriverIOについての講演の一環として、イベント自体や会社から支払われないカンファレンスやミートアップへの旅行のための航空券やホテル宿泊費を申請する資格があります。最大$500まで申請できます。旅費の払い戻し要件は以下の通りです：

- イベント終了後、メインのソーシャルメディアアカウント（𝕏、LinkedInまたは個人ブログなど）からコレクティブのすべての貢献者に感謝する投稿を送信する必要があります。
- イベントへの地上または航空輸送、ホテル宿泊などの適格な旅費に資金を使用する必要があります。
- 払い戻しリクエストには領収書を添付する必要があります。
