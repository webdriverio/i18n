TestLensはDevTools UIとテストソースファイルの間でインテリジェントなコードナビゲーションを提供します。UI上の任意のテストをクリックすると、コードエディタ内のそのテストの定義に正確な行番号を含めて直接ジャンプできます。

**仕組み：**
- **視覚的なテスト階層** - テスト構造（スイート、テスト、シナリオ）をツリービューで表示
- **ソースマッピング** - 各テストはソースファイル内の正確な位置にリンク
- **ワンクリックナビゲーション** - どのテストでもクリックするとエディタで開く
- **エディタ統合** - file:lineプロトコルを通じて人気のコードエディタで動作

UIとコード間のこのシームレスな接続により、テスト失敗を調査する際の手動コード検索を排除し、デバッグを迅速化します。
