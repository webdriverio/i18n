title: دعم الإطارات المتعددة
---

تعمل DevTools تلقائيًا مع Mocha وJasmine وCucumber دون الحاجة إلى أي تكوين خاص بإطار العمل. ما عليك سوى إضافة الخدمة إلى تكوين WebDriverIO الخاص بك وستعمل جميع الميزات بسلاسة بغض النظر عن إطار الاختبار الذي تستخدمه.

**الإطارات المدعومة:**
- **Mocha** - تنفيذ على مستوى الاختبار والمجموعة مع تصفية grep
- **Jasmine** - تكامل كامل مع تصفية قائمة على grep
- **Cucumber** - تنفيذ على مستوى السيناريو والأمثلة مع استهداف feature:line

واجهة التصحيح نفسها، وإعادة تشغيل الاختبار، وميزات التصور تعمل بشكل متسق عبر جميع الإطارات.

## التكوين

```js
// wdio.conf.js
export const config = {
    framework: 'mocha', // or 'jasmine' or 'cucumber'
    services: ['devtools'],
    // ...
};
```
