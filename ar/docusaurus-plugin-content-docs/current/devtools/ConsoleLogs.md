title: سجلات وحدة التحكم
---

التقاط وفحص جميع مخرجات وحدة تحكم المتصفح أثناء تنفيذ الاختبار. تسجل أدوات المطور رسائل وحدة التحكم من تطبيقك (`console.log()`، `console.warn()`، `console.error()`، `console.info()`، `console.debug()`) بالإضافة إلى سجلات إطار WebDriverIO استنادًا إلى `logLevel` المكون في ملف `wdio.conf.ts` الخاص بك.

**الميزات:**
- التقاط رسائل وحدة التحكم في الوقت الفعلي أثناء تنفيذ الاختبار
- سجلات وحدة تحكم المتصفح (log، warn، error، info، debug)
- سجلات إطار WebDriverIO مفلترة حسب `logLevel` المكون (trace، debug، info، warn، error، silent)
- الطوابع الزمنية التي توضح بالضبط متى تم تسجيل كل رسالة
- عرض سجلات وحدة التحكم جنبًا إلى جنب مع خطوات الاختبار ولقطات شاشة المتصفح للسياق

**التكوين:**
```js
// wdio.conf.ts
export const config = {
    // مستوى تفصيل التسجيل: trace | debug | info | warn | error | silent
    logLevel: 'info', // يتحكم في سجلات الإطار التي يتم التقاطها
    // ...
};
```

هذا يجعل من السهل تصحيح أخطاء JavaScript، وتتبع سلوك التطبيق، ورؤية العمليات الداخلية لـ WebDriverIO أثناء تنفيذ الاختبار.
