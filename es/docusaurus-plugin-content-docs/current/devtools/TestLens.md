TestLens proporciona navegación inteligente de código entre la interfaz de usuario de DevTools y tus archivos fuente de prueba. Haz clic en cualquier prueba en la interfaz de usuario para saltar directamente a la definición de esa prueba en tu editor de código, completo con números de línea exactos.

**Cómo funciona:**
- **Jerarquía visual de pruebas** - Mira la estructura de tus pruebas (suites, pruebas, escenarios) en una vista de árbol
- **Mapeo de fuentes** - Cada prueba se vincula a su ubicación exacta en tus archivos fuente
- **Navegación con un clic** - Haz clic en cualquier prueba para abrirla en tu editor
- **Integración con editor** - Funciona con editores de código populares a través de protocolos file:line

Esta conexión sin problemas entre la interfaz de usuario y el código acelera la depuración al eliminar la búsqueda manual de código cuando se investigan fallos en las pruebas.
