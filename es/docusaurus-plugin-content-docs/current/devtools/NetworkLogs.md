title: Registros de Red
---

Monitorea e inspecciona toda la actividad de red durante tus pruebas. DevTools captura cada solicitud y respuesta HTTP, brindándote visibilidad completa de las llamadas a API, la carga de recursos y el tiempo de red - justo como las DevTools del navegador.

**Lo que se captura:**
- **Detalles de solicitud** - URL, método, encabezados, parámetros de consulta, cuerpo de la solicitud
- **Datos de respuesta** - Código de estado, encabezados de respuesta, cuerpo de respuesta, tiempo
- **Tipos de recursos** - Solicitudes XHR/Fetch, scripts, hojas de estilo, imágenes y más
- **Métricas de rendimiento** - Tiempo de solicitud, duración y cascada de red

Esto es invaluable para depurar problemas de API, identificar solicitudes lentas, validar cargas de datos y comprender el comportamiento de red de tu aplicación durante las pruebas.
