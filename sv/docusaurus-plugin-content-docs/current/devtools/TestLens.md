TestLens ger intelligent kodnavigering mellan DevTools-gränssnittet och dina testkällfiler. Klicka på vilket test som helst i användargränssnittet för att hoppa direkt till testets definition i din kodredigerare, komplett med exakta radnummer.

**Hur det fungerar:**
- **Visuell testhierarki** - Se din teststruktur (sviter, tester, scenarier) i en trädvy
- **Källkartläggning** - Varje test länkar till dess exakta plats i dina källfiler
- **Navigering med ett klick** - Klicka på vilket test som helst för att öppna det i din redigerare
- **Redigerarintegration** - Fungerar med populära kodredigerare genom file:line-protokoll

Denna sömlösa koppling mellan användargränssnitt och kod påskyndar felsökning genom att eliminera manuell kodsökning när du undersöker testmisslyckanden.
