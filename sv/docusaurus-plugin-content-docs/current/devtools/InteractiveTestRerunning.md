title: Interaktiv testomkörning och visualisering
---

Se dina tester köras i realtid med livepreviews av webbläsaren och automatiska skärmbilder som tas efter varje WebDriver-kommando. Användargränssnittet visar en komplett visuell tidslinje för din testkörning, som visar webbläsarens exakta tillstånd vid varje steg.

Efter att din första testkörning är klar kan du klicka på vilket testfall eller testsuite som helst i DevTools UI för att köra om det direkt. Detta eliminerar behovet av att starta om hela testrunner eller ändra i din kod, vilket dramatiskt snabbar upp felsökningscykler.

**Nyckelfunktioner:**
- **Realtidsvisualisering** - Automatisk skärmbildstagning efter varje kommando (click, toBeExisting, etc.)
- **Liveförhandsvisning av webbläsare** - Se din applikations aktuella tillstånd under testkörningen
- **Kommandotidslinje** - Se detaljerade loggar med tidsstämplar för varje åtgärd
- **Interaktiv omkörning** - Klicka på valfritt test eller testsuite för att köra om det direkt
- **Körning i samma session** - Tester körs om i samma webbläsarsession för snabbare återkoppling
- **Scrollbar historik** - Granska vilken punkt som helst under testkörningen
