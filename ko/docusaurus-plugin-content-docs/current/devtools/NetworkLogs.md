테스트 중 모든 네트워크 활동을 모니터링하고 검사하세요. DevTools는 모든 HTTP 요청과 응답을 캡처하여 브라우저 DevTools처럼 API 호출, 리소스 로딩, 네트워크 타이밍에 대한 완벽한 가시성을 제공합니다.

**캡처되는 내용:**
- **요청 세부 정보** - URL, 메소드, 헤더, 쿼리 매개변수, 요청 본문
- **응답 데이터** - 상태 코드, 응답 헤더, 응답 본문, 타이밍
- **리소스 유형** - XHR/Fetch 요청, 스크립트, 스타일시트, 이미지 등
- **성능 지표** - 요청 타이밍, 지속 시간, 네트워크 워터폴

이는 API 문제 디버깅, 느린 요청 식별, 데이터 페이로드 검증, 테스트 중 애플리케이션의 네트워크 동작 이해에 매우 유용합니다.
