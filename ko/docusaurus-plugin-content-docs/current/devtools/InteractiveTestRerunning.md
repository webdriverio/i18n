실시간 브라우저 미리보기와 각 WebDriver 명령 후 자동으로 캡처된 스크린샷으로 테스트 실행을 실시간으로 확인하세요. UI는 테스트 실행의 전체 시각적 타임라인을 표시하여 각 단계에서의 정확한 브라우저 상태를 보여줍니다.

초기 테스트 실행이 완료된 후, DevTools UI에서 테스트 케이스나 스위트를 클릭하여 즉시 재실행할 수 있습니다. 이렇게 하면 전체 테스트 러너를 다시 시작하거나 코드를 수정할 필요가 없어, 디버깅 사이클이 크게 단축됩니다.

**주요 기능:**
- **실시간 시각화** - 모든 명령(click, toBeExisting 등) 후 자동 스크린샷 캡처
- **실시간 브라우저 미리보기** - 테스트 실행 중 애플리케이션의 현재 상태 확인
- **명령 타임라인** - 각 작업에 대한 타임스탬프가 포함된 상세 로그 보기
- **인터랙티브 재실행** - 테스트나 스위트를 클릭하여 즉시 재실행
- **동일 세션 실행** - 더 빠른 피드백을 위해 동일한 브라우저 세션에서 테스트 재실행
- **스크롤 가능한 히스토리** - 테스트 실행의 어느 시점이든 검토 가능
