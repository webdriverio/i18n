테스트렌즈는 DevTools UI와 테스트 소스 파일 사이에 지능적인 코드 탐색 기능을 제공합니다. UI에서 아무 테스트나 클릭하면 정확한 라인 번호와 함께 코드 에디터에서 해당 테스트의 정의로 바로 이동할 수 있습니다.

**작동 방식:**
- **시각적 테스트 계층 구조** - 트리 뷰에서 테스트 구조(스위트, 테스트, 시나리오)를 확인
- **소스 매핑** - 각 테스트는 소스 파일의 정확한 위치에 연결
- **원클릭 탐색** - 아무 테스트나 클릭하면 에디터에서 열림
- **에디터 통합** - file:line 프로토콜을 통해 인기 있는 코드 에디터와 작동

UI와 코드 간의 이러한 원활한 연결은 테스트 실패를 조사할 때 수동 코드 검색을 제거하여 디버깅 속도를 높입니다.
