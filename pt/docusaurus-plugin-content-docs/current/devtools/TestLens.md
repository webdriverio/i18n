O TestLens oferece navegação inteligente de código entre a interface de usuário do DevTools e seus arquivos de teste de origem. Clique em qualquer teste na interface para ir diretamente à definição desse teste no seu editor de código, com números de linha exatos.

**Como Funciona:**
- **Hierarquia visual de testes** - Veja a estrutura de seus testes (suítes, testes, cenários) em uma visualização em árvore
- **Mapeamento de origem** - Cada teste está vinculado à sua localização exata nos seus arquivos de origem
- **Navegação com um clique** - Clique em qualquer teste para abri-lo em seu editor
- **Integração com editor** - Funciona com editores de código populares através de protocolos file:line

Esta conexão perfeita entre interface e código acelera a depuração, eliminando a busca manual de código ao investigar falhas nos testes.
